QQKRLS

Quantile-on-Quantile Kernel-Based Regularized Least Squares
A Python library for nonparametric econometric analysis with publication-quality visualizations

📦 PyPI v1.1.1 ⭐ GitHub 🚀 Get Started

✨ Key Features

Four powerful econometric methods in one unified Python package

🔬

KRLS

Kernel Regularized Least Squares — nonparametric regression with automatic bandwidth selection and LOO cross-validation

📊

QQKRLS

Quantile-on-Quantile KRLS — nonparametric marginal effects across all distributional quantile pairs with bootstrap inference

🎨

Visualization Suite

Publication-quality heatmaps, 3D surfaces, contour plots, and p-value maps with MATLAB-style aesthetics

📋

LaTeX Tables

Auto-generated LaTeX tables, descriptive statistics, and CSV export — ready for top-tier journals

🧪

Diagnostics

Full pre- and post-estimation test battery: BDS, Ramsey RESET, Breusch-Pagan, Jarque-Bera, Andrews stability

⚡

Production-Ready

Clean API, comprehensive documentation, type hints, and pip-installable from PyPI

📖 Interpretation Guide

How to read and interpret QQKRLS results for your research paper

🗺️ Heatmap Axes
AxisMeaning
X-axis (θ)Quantile of the independent variable (0.05 – 0.95)
Y-axis (τ)Quantile of the dependent variable (0.05 – 0.95)
ColorMarginal effect coefficient at (θ, τ)
Stars*** p<0.01, ** p<0.05, * p<0.10
🔍 Key Patterns
PatternInterpretation
Uniform colorHomogeneous effect across quantiles
Color gradientEffect varies systematically
Dark cornersStrong effects at extremes
White bandNo significant effect at those pairs
🔬 Methodological Justification Workflow
StepTestPurpose
1Ramsey RESETReject linearity → justifies KRLS over OLS
2BDSReject i.i.d. → justifies nonparametric approach
3Breusch-PaganReject homoskedasticity → justifies quantile analysis
4KRLS R²Confirm good nonparametric model fit
5QQKRLS heatmapReveal heterogeneous distributional effects
